INTRODUCTION
------------

The Views Material Design Lite module provides several MDL components as views 
display plugins and field handlers.


REQUIREMENTS
------------

The module depends on the views module and the components will not take effect
without the inclusion of the Material Design Lite's css framework, preferably
through the theme. 


INSTALLATION
------------

Just enable the module from the modules page or using drush pm-enable


CONFIGURATION
-------------

Any view will have the components available once the module is enabled.
